# Python Practice for Data Engineering

Daily, hands-on notebooks to build Python fundamentals and data-engineering-ready skills.

---

## What’s in this repo
This repository is a growing collection of Jupyter notebooks where I practice core Python concepts with a data engineering mindset.

### Current notebooks
- **`numpy.ipynb`** — NumPy fundamentals and array operations
- **`string and controlflow.ipynb`** — Strings, conditionals, loops, and control flow
- **`oops.ipynb`** — Object-Oriented Programming (classes, methods, inheritance)

---

## Goals
- Build strong Python fundamentals
- Practice consistently (every day)
- Apply concepts in a way that maps to data engineering work (ETL, data processing, quality checks, automation)

## How to run locally

### 1) Create a virtual environment
```bash
python -m venv .venv
```

### 2) Activate it
- **macOS / Linux**
  ```bash
  source .venv/bin/activate
  ```
- **Windows (PowerShell)**
  ```powershell
  .venv\Scripts\Activate.ps1
  ```

### 3) Install Jupyter
```bash
pip install --upgrade pip
pip install notebook
```

### 4) Start Jupyter
```bash
jupyter notebook
```
